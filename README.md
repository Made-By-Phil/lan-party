# LAN Party

An opinionated backbone for hosting quick local-multiplayer browser games on your LAN.
One computer hosts; everyone joins from their phone or laptop browser. Games are
drop-in folders of TypeScript.

```
npx lan-party
```

Then open the printed URL (or scan the QR code) from any device on the same network.

Status: under construction. See [docs/DESIGN.md](docs/DESIGN.md) for the architecture
and SDK contract, [docs/decisions.md](docs/decisions.md) for the decision log, and
[BACKLOG.md](BACKLOG.md) for deferred ideas.
